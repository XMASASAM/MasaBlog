Hugoの使い方メモ

10月 28, 2024

Hugoの使い方をメモしていく。

注意点

Windowsの場合は PowerShellを使う必要がある。 Windows PowerShell では正常に動作しない場合がある。(PowerShellとWindows PowerShellは別物)

ポストを追加

hugo new content content/posts/post-X/index.md

実行

hugo server

css追加(anankeの場合)

$HugoProject$/hugo.toml
...

[params]
    custom_css = ["custom.css"]
$HugoProject$/assets/ananke/css/custom.css
cssを書く

custom.cssの箇所はどんな名前でもいい。 ファイルを追加したい場合は

$HugoProject$/hugo.toml
...

[params]
    custom_css = ["custom.css","custom2.css"]

$HugoProject$/assets/ananke/css/custom2.css　で追加でOK。
